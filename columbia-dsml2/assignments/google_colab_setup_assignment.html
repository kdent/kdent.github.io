Prepare Your Development Environment

Create a folder on your Google drive where you will store data files to use during class. You may be asked to sign in to your Google account if you are not signed in already.

  1. Go to https://drive.google.com/drive/my-drive From there create a new folder called "columbia-ml-data", and navigate into that directory.
  2. Download the file iris.csv to your local machine and then upload it to the columbia-ml-data directory you just created on your Google drive.
  3. Next, go to https://colab.research.google.com/ and create a new notebook.
  4. Write the Python code in your notebook to
    1. Open the file
    2. Count the number of lines in the file
    3. Print the number of lines

If you need help with the code to count the number of lines, click the Show Code button.
