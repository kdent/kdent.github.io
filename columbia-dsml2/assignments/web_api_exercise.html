Exercise: Using a Web API to Retrieve Museum Data

Overview

In this exercise, you will learn how to use a web API to retrieve data from an online source. You will connect to the Art Institute of Chicago's public API, download information about artwork in the museum's collection, and search the returned data to find a specific artwork.

This exercise introduces several important data science skills:

  • Making requests to a web API
  • Working with JSON data
  • Storing data in arrays
  • Looping through data
  • Searching datasets for specific information

Background: What is an API?

A Web API (Application Programming Interface) allows programs to communicate with other programs and retrieve data. Instead of manually downloading a dataset, your program can ask a server for the data it needs.

The Art Institute of Chicago provides a public API that allows developers to access information about artworks in its collection.

API Documentation: https://api.artic.edu/docs/

Your Goal

Use the Art Institute of Chicago API to retrieve a list of artwork objects. Store the returned objects in an array. Search through the array until you find the artwork titled "Two Itinerant Musicians". Print the JSON information for that artwork.

Step 1: Retrieve Data from the API

Use the following API endpoint: 

https://api.artic.edu/api/v1/artworks?page=5&limit=100

This request asks the API for:

  • Page 5 of the artwork collection
  • 100 artwork records

When you access this URL, the API returns data in JSON format.

Step 2: Store the Artwork List

The JSON response contains an array of artwork objects. Create a variable to store this array.

Your program should contain code similar to: 

json_resp = response.json()
artworks = json_resp["data"]

Now your program has a list of artwork objects that it can search through. Each item in the list is a dictionary with many attributes about the artwork it represents. One of the attributes is the 'title' of the work.

Step 3: Search the Array

Traverse the array using a loop. For each artwork object:

  1. Look at the artwork title
  2. Check whether the title is "Two Itinerant Musicians"
  3. If you find it, print the complete JSON object

Your logic should look something like: 

for artwork in artworks:

    if artwork title is "Two Itinerant Musicians":

        print artwork information

Step 4: Examine the Result

After finding the artwork, examine the JSON output. Look for information such as:

  • Artist
  • Date created
  • Medium
  • Description
  • Artwork ID

Challenge Questions

  1. How many artwork objects did the API return?
  2. What information about the artwork would be useful for a museum search application?
  3. Modify your program so that it searches for a different artwork title.
  4. Modify your program to print the titles of all artworks in the returned dataset.

Extension: Connect to Data Science

Large datasets are often accessed through APIs. The same skills you practiced here are used when collecting data for machine learning projects, dashboards, and data analysis.

In a real data science workflow:

  • APIs provide the raw data
  • Programs clean and organize the data
  • Machine learning models analyze patterns in the data
