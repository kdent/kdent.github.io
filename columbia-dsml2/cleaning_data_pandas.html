Assignment: Cleaning Data with Pandas

For this assignment you will practice loading a new data set from New York State elections. There are duplicated rows and some other problems with the data that need to be cleaned up. Follow the instructions below:

  1. Start a new Colab notebook
  2. Load the data set 20221108_ny_general_albany_precinct.csv into a Pandas data frame.
  3. Find and remove and duplicate rows
  4. Find "precinct" values that contain extraneous asterisk characters ('*') and remove the asterisk.
  5. Write your cleaned up data frame to a new file.
